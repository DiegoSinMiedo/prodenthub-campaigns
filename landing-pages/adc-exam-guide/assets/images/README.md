# Campaign Images

Place campaign-specific images in this folder.

## Required Image

### `guide-cover.jpg`
- **Purpose:** Preview image of the PDF guide cover shown in the form
- **Dimensions:** A4 ratio (1:1.414)
  - Recommended: 595px × 842px (A4 at 72 DPI)
  - Or: 1200px × 1697px (for retina displays)
- **Format:** JPG or PNG
- **File size:** Keep under 200KB for fast loading
- **Content:** The cover page of your "Cracking Clinical Cases" PDF guide

## How to add your image:

1. Export the first page of your PDF as an image
2. Resize to recommended dimensions (maintaining A4 ratio)
3. Optimize for web (compress to under 200KB)
4. Save as `guide-cover.jpg` in this folder
5. The image will automatically appear on the landing page

## Image styling:

The image is styled with:
- Maximum width: 240px (mobile-friendly)
- Maintains A4 aspect ratio (1:1.414)
- Rounded corners with shadow
- Centered in the form
- Responsive (scales down on mobile)
